# Pomodoro Timer

Приложение для управления временем по методу Pomodoro, разработанное на React Native с архитектурой FSD.

## Функциональность

- **Настраиваемый таймер** работы/отдыха с дефолтными значениями (25/5/15 минут)
- **Автоматическое переключение** между сессиями работы и отдыха
- **Встроенная фоновая музыка** (3 трека для концентрации)
- **Статистика** завершенных сессий и общего времени работы
- **Гибкие настройки** интервалов и количества сессий
- **Сохранение данных** между сессиями (AsyncStorage)
- **Кроссплатформенность** - IOS и Android

## Быстрый старт

### Предварительные требования

- Node.js (версия 16 или выше)
- npm или yarn
- Expo CLI (`npm install -g expo-cli`)
- Expo Go на мобильном устройстве (для тестирования)

### Установка

```bash
# Клонируйте репозиторий
git clone https://github.com/aniizif/pomodoro-app.git

# Перейдите в папку проекта
cd pomodoro-app

# Установите зависимости
npm install

# Запустите приложение
npm start
```

## Запуск на устройстве

Установите Expo Go из App Store или Google Play

Отсканируйте QR-код из терминала или Expo Dev Tools

Приложение запустится на вашем устройстве

## Архитектура проекта
Проект построен по методологии Feature-Sliced Design (FSD):
```
src/
├── app/          # Инициализация приложения, провайдеры
│   └── providers/
│       └── with-navigation.js    # Настройка навигации
├── pages/        # Страницы приложения
│   ├── home/     # Главная страница с таймером
│   ├── settings/ # Страница настроек
│   └── statistics/ # Страница статистики
├── features/     # Фичи (бизнес-логика)
│   ├── timer/    # Логика таймера Pomodoro
│   └── music/    # Управление фоновой музыкой
├── entities/     # Бизнес-сущности
│   └── session/  # Модель сессии Pomodoro
└── shared/       # Переиспользуемые компоненты и утилиты
    ├── ui/       # UI компоненты (TimerDisplay, ControlButtons, MusicControl)
    └── lib/      # Утилиты (форматирование времени, storage, notifications)
```

## Как использовать
Запуск таймера: Нажмите "Старт" для начала рабочей сессии

Пауза: Нажмите "Пауза" чтобы приостановить таймер

Сброс: Кнопка "Сброс" возвращает таймер в начальное состояние

Настройки: Настройте длительность работы, короткого и длинного отдыха

Статистика: Отслеживайте количество завершенных сессий и общее время

Музыка: Выберите фоновый трек для лучшей концентрации

## Цикл Pomodoro
Работа (25 минут по умолчанию)

Короткий отдых (5 минут)

Повторите 4 раза

Длинный отдых (15 минут)

## Добавление своей музыки
Поместите MP3 файлы в папку assets/music/

Обновите конфигурацию в src/features/music/model.js:
```js
const SOUNDS = {
  yourTrack: {
    name: 'Название трека',
    file: require('../../assets/music/your-track.mp3'),
  },
  // ... другие треки
};
```
